# Eden Lane Brief — Photo Sourcing Guide

29 photos needed. Save each one as a JPEG with the **exact filename listed** into the `photos/` folder of your repo.

**Target image size:** 800–1200px wide. Bigger images should be resized (instructions at bottom).

## How to use this guide

For each plant:

1. Click the link in the table — it opens the Missouri Botanical Garden plant detail page (or a short search-result list with one click to the plant)
2. The lead photo appears at the top left under the plant name
3. Click that small thumbnail — it opens a larger version in a new window
4. **Right-click the larger version → Save Image As**
5. Save with the **exact filename** shown in the table, into your `photos/` folder

Some pages have a "More Images" section near the bottom with multiple landscape views. Pick whichever shows the plant best as a screening / hedge / specimen — that's what board members care about.

## Workflow tips

- **Open multiple tabs.** Right-click each link → "Open Link in New Tab" so you can chew through them in batches.
- **Cmd+S / Ctrl+S won't work** — these site pages have lots of layout. Right-click the actual image only.
- **If a search returns multiple plants** (like clicking the Acer rubrum link might list several cultivars), pick the straight species — it's labeled simply "*Acer rubrum*" without a quoted cultivar name afterward.
- **Some plants will be cultivars** (Ivory Halo, etc.). The search should land you directly on the right cultivar page.
- **If MBG doesn't have a great photo** for a particular plant, fall back to:
  - **USDA NRCS PLANTS Database** (https://plants.usda.gov) — public domain, especially good for native species
  - **NC State Extension Plant Toolbox** (https://plants.ces.ncsu.edu)
  - **UConn Plant Database** (https://plantdatabase.uconn.edu)
  - **Wikimedia Commons** (https://commons.wikimedia.org/wiki/Main_Page) — filter by Creative Commons license

## Bulk resize after downloading

Most MBG and Proven Winners photos will be reasonable sizes (under 1MB), but a few may be larger. For consistency and fast page loads:

**Online (no install, any OS):**
- https://bulkresizephotos.com — drag in all 29, set max width 1200px, JPEG quality 80, download zip

**Mac:**
- Select all in Finder → right-click → Quick Actions → Convert Image → JPEG, Image Size: Custom 1200px

**Windows:**
- Microsoft PowerToys → Image Resizer → Custom 1200px max width

**Target:** each photo under 400 KB. Total `photos/` folder under 12 MB.
